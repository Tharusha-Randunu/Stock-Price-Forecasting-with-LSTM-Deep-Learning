## 📈 Stock Price Forecasting with LSTM Deep Learning

## 📌 Project Overview
A deep learning-based stock market forecasting and investment recommendation system developed as part of the **Advanced Machine Learning Applications** module at the University of Moratuwa.
The project combined **time-series forecasting** with **equity research methodologies** to analyze and predict the performance of leading enterprise software companies, enabling data-driven investment decision-making.

---

## 🛠️ Features

### 📥 Data Collection & Preprocessing
- Collected five years of historical stock price data for **Oracle, IBM, and SAP**
- Used financial APIs and Python for data extraction and cleaning

### 🔎 Exploratory Data Analysis
- Technical indicators and volatility analysis
- Drawdown metrics
- Interactive financial visualizations

### 📰 Market Context Analysis
- Investigated real-world market events influencing stock performance
- Integrated findings into an investor-focused research analysis

### 🤖 Model Design & Training
- Designed and trained stacked **LSTM neural network** models
- Applied regularization techniques to improve generalization

### ⚙️ Hyperparameter Optimization
- Tuned LSTM units, dropout rates, and learning rates
- Used **Keras Tuner** for automated optimization

### 🔄 Model Evaluation & Refinement
- Iterative experimentation across multiple stocks
- Performance assessment and model refinement

### 🔮 Forecasting
- Generated multi-step forecasts for upcoming trading periods using trained models

### 📄 Equity Research Reporting
- Developed a professional equity research report
- Provided **Buy, Hold, and Sell** recommendations backed by predictive analytics and market insights

---

## 🚀 Technologies Used
- **Python** – Data processing and model development
- **Keras / TensorFlow** – LSTM model design and training
- **Keras Tuner** – Hyperparameter optimization
- **Financial APIs** – Historical stock data collection
- **Visualization Libraries** – Technical and financial charting

---

## 🎓 Project Context
Developed as part of the Advanced Machine Learning Applications module at the University of Moratuwa, applying deep learning techniques to real-world equity research and investment forecasting.
